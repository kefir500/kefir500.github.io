Description

APK Icon Editor is an APK editor designed to easily change Android app icon, name, version, images, strings and other resources. Available for Windows, OS X and Linux.

  • Edit APK, extract and change APK resources.
  • Change Android app icon, name and version.
  • Sign and optimize Android applications.
  • Supported image formats: PNG, ICO, GIF, JPEG, BMP.
  • Size presets for devices: Android, BlackBerry, Amazon Kindle Fire.
  • Cloud storage upload: Dropbox, Google Drive, Microsoft OneDrive.
  • Multilingual interface.

Requirements

  • JRE 32-bit (for most users) – to sign APK files and use Apktool.
  • JDK 32-bit (optional alternative) – to sign APK files with KeyStore.
