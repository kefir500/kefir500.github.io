Compare which version number is greater.

This function checks if v1 is greater then v2. Note that %1.%2.%3.N scheme is used to represent application version. Only numbers and dots are acceptable.

Return values
\cTRUE if v1 > v2.
\cFALSE if v1 <= v2.
