Detailed Description

This class provides profiles with DPI presets for different devices.

Profile class provides presets of DPI sizes for different devices. List of available profiles is retrieved using init static method.

Concept: Preset describes single DPI size. Profile stores multiple presets (for LDPI, MDPI, etc.).
