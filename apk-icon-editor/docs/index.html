Structure

MainWindow is the central object. It provides connections between the Apk module and the DrawArea widget.
Apk class provides packing and unpacking APK files and loads Icon list.
DrawArea widget draws the single Icon currently selected in the ComboList widget.

Summary: According to the selected Profile in the ComboList widget, MainWindow gets the appropriate Icon from Apk and passes it to the DrawArea widget.
