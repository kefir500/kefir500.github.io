APK Editor Studio

APK Editor Studio — это мощный и одновременно простой в использовании инструмент для декомпиляции APK. Распаковывайте, редактируйте и заменяйте ресурсы, изображения и значки APK. Добавляйте новые переводы, создавайте ваши собственные модификации APK или изучайте их внутреннюю структуру. Множество полезных инструментов помогут вам легко и просто заменить значок и имя приложения, автоматически подписать APK и установить его на ваше устройство.

Возможности

Редактор значков

APK Editor Studio автоматически определяет значки APK, благодаря чему вы можете изменить иконку приложения за считанные секунды.

Редактор имени приложения

Вы можете легко менять имя приложения Android, в том числе и его переводы на различные языки.

Редактор изображений

Просмотр и замена изображений. Автоматическая конвертация между PNG, SVG, JPEG, BMP, ICO, GIF и многими другими форматами.

Редактор кода

Встроенный редактор текста с подсветкой синтаксиса для форматов XML и YAML.

Инспектор ресурсов

Интеллектуальный инспектор ресурсов группирует ресурсы APK по их квалификаторам для более удобного просмотра.

Менеджер проекта

Центральный хаб для редактирования APK: быстрый доступ к часто используемым действиям и ключевым функциям.

Редактор разрешений

Добавляйте и удаляйте разрешения Android без необходимости редактировать манифест Android вручную.

Редактор манифеста

Редактируйте наиболее важные свойства манифеста Android-приложения без необходимости разбирать его внутреннюю структуру.

Проводник Android

Управляйте файлами на вашем устройстве Android и делайте снимки экрана прямо с вашего компьютера.

Подпись APK

Автоматическая подпись APK — с использованием как ключа по умолчанию, так и вашего собственного — через менеджер ключей.

Установка APK

Установка APK на ваши устройства в один клик. Простое управление несколькими устройствами, быстрый доступ.

Оптимизация APK

После упаковки APK автоматически оптимизируется для повышения стабильности работы и снижения потребления оперативной памяти.
